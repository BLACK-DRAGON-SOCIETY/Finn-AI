# 💹 Financial AI Guide

An intelligent, modern web-based financial assistant designed to provide **stock analysis, market trends, company insights, and financial guidance** through an interactive AI-powered chat interface. Built with a sleek dark-mode dashboard, responsive UI, markdown-rendered conversations, and real-time backend integration for financial analysis. :contentReference[oaicite:0]{index=0} :contentReference[oaicite:1]{index=1}


## 🚀 Features

### 📊 AI-Powered Financial Assistance
- Stock price analysis
- Market trends & insights
- Company performance metrics
- Crypto and portfolio discussions
- Financial Q&A in chat format

### 🎨 Modern User Interface
- Elegant dark mode by default
- Responsive Tailwind CSS design
- Sidebar with recent chats
- Animated loading indicators
- Markdown & syntax-highlighted responses
- Professional dashboard layout

### 💬 Smart Chat Experience
- User and AI conversation bubbles
- Real-time API integration
- Markdown support using `marked.js`
- Code block highlighting with `highlight.js`
- Smooth scrolling and dynamic UI updates
